# Recruiter Notes

This project was built to show practical junior IT skills, not only programming ability.

## Skills shown

- Understanding of helpdesk workflows
- User support and ticket documentation
- Device inventory tracking
- Basic SLA/due date logic
- Authentication and role-based visibility
- Dashboard metrics
- CSV reporting
- Clean documentation
- Basic automated tests

## Good interview talking points

- Why a ticketing system needs clear categories, priorities, and status history
- How support technicians document troubleshooting steps
- Why assets should be linked to support tickets
- How CSV export helps managers report workload
- Why separating `services.py` and `selectors.py` improves maintainability
